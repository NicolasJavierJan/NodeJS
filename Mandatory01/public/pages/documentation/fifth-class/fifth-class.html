Importing and Exporting

There are two different ways to import and export: NodeJS (require) and ES6 (import and export).

"Require":

const express = require("express");

We use "require" to include modules (JavaScript libraries) into our application.

In our "Tanks Museum" project, we used the "require" method to import and export data from different files. For example, in our "app.js" file:

const { getTanks, addTank } = require("./util/tanks.js")

In this way, we are able to use the getTanks and addTank function that are present in "tanks.js":

function getTanks(){

return tanks;

}

function addTank(tank) {

tanks.push(tank);

return tanks;

}

Import and Export:

To use this way, we have to edit the "package.json" file:

"type": "module",

This will give us access to the Import and Export keywords. There are two types of imports and exports: named and default.

Named:

export const guards = [];


import { guards } from "./barracks.js";

And default:

export default guards


import guards from "./barracks.js";

We can also import scripts in the HTML with the Script tag. We can import more than one file, and that would effectively create a big .js file, making it risky (we could risk using the same variable name in different files, for example).
